# 贡献

> 了解如何为 Nuxt Color Mode 做出贡献

欢迎贡献！以下是您可以帮助改进 Nuxt Color Mode 的方法。

## 开发环境搭建

1. 克隆仓库：

```bash
git clone https://github.com/nuxt-modules/color-mode.git
cd color-mode
```

1. 安装依赖：

```bash
pnpm install
```

1. 启动开发服务器：

```bash
pnpm dev
```

这将在 `http://localhost:3000` 启动 playground。

## 项目结构

- `src/` - 模块源代码
- `playground/` - 开发测试 playground
- `docs/` - 文档网站
- `test/` - 测试套件

## Pull Requests

1. Fork 仓库
2. 为您的功能或修复创建新分支
3. 进行更改
4. 如适用，添加测试
5. 运行测试套件：`pnpm test`
6. 遵循 [Conventional Commits](https://www.conventionalcommits.org/) 提交更改
7. 提交 Pull Request

## 问题反馈

发现 BUG 或有功能请求？请在 [GitHub](https://github.com/nuxt-modules/color-mode/issues) 上提交 issue。
