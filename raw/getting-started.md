# 介绍

> 使用 Nuxt 轻松实现带自动检测的暗黑与亮色模式 🌗

使用 Nuxt 轻松实现带自动检测的暗黑与亮色模式 🌗

## 特性

- 向 `<html>` 添加可配置的 `.${color}` 类，方便 CSS 主题定制
- 强制页面使用特定的颜色模式（适合渐进式开发）
- 支持客户端渲染和通用渲染
- 与 [@nuxtjs/tailwindcss](https://github.com/nuxt-modules/tailwindcss) 开箱即用
- 自动检测系统的 [color-mode](https://drafts.csswg.org/mediaqueries-5/#descdef-media-prefers-color-mode)
